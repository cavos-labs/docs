Cavos provides invisible wallet infrastructure for Starknet applications. Users authenticate with familiar methods (Google, Apple, or Passkeys) and get a fully functional wallet without managing seed phrases or installing extensions.

## Key Features

<CardGroup cols={2}>
  <Card title="Passkey Security" icon="fingerprint">
    Private keys are encrypted with device biometrics (FaceID/TouchID) and never leave the user's device unencrypted.
  </Card>
  <Card title="Gasless Transactions" icon="bolt">
    Users can transact without holding ETH. All gas fees are sponsored via AVNU Paymaster.
  </Card>
  <Card title="Cross-Platform" icon="devices">
    SDKs for both web (React) and mobile (React Native) with unified APIs.
  </Card>
  <Card title="OAuth Integration" icon="key">
    Support for Google and Apple sign-in with automatic wallet creation.
  </Card>
</CardGroup>

## How It Works

1. **User authenticates** with Google, Apple, or a passkey
2. **Wallet is created** automatically with an ArgentX-compatible account
3. **Wallet is deployed** gaslessly using AVNU Paymaster
4. **Transactions execute** without users needing to hold ETH

## Quick Example

```tsx
import { CavosProvider, useCavos } from '@cavos/react';

function App() {
  return (
    <CavosProvider config={{ appId: 'your-app-id' }}>
      <WalletButton />
    </CavosProvider>
  );
}

function WalletButton() {
  const { login, address, execute } = useCavos();

  if (address) {
    return <p>Connected: {address}</p>;
  }

  return <button onClick={() => login('google')}>Login with Google</button>;
}
```
